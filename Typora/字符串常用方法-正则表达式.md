### 使用正则表达式的方法

| API       | 描述                       | 参数                                    | 返回值                             |
| --------- | -------------------------- | --------------------------------------- | ---------------------------------- |
| match()   | 根据参数返回匹配结果       | String \| RegExp                        | Array                              |
| replace() | 将匹配结果替换为指定字符串 | [ String \| RegExp，String \| Function] | String(替换后的)                   |
| search()  | 找出第一个匹配项的索引     | String \| RegExp                        | 首次匹配的位置（成功）\| -1 (失败) |
| split()   | 根据传入的分隔符切割源文本 | String \| RegExp                        | Array                              |
